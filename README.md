# Battleship

Battleship built using TDD testing and webpack.<br> 
HTML page set up, <br>
Javascript with classes for ship, player, and gameboard. <br>
Single player game versus computer, interactive ship placement, and turn-based gameplay. <br>
CSS for styling. Project of The Odin Project. <br>

How to Play:

First place your ships.  <br>
Click and drag to the Player Board. <br>
You can click "Flip" to make your ships vertical.  <br>
Once all ships are placed click "Start". <br>
Select a cell on the Computer Board to attack. <br>
Wait until the computer finishes attacking and select a cell again. <br>
Continue this until the computer or you sinks all the opponents ships. <br>
At any time you can click "Reset" to reset the game.
